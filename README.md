Network Traffic Analyzer for VMs
This lab demonstrates the ability to capture, filter, and analyze live network traffic generated by an Ubuntu Linux virtual machine using Wireshark.

1. Lab Objective
Configure a virtual environment to allow for external network communication.

Successfully install and configure the Npcap packet capture driver on the host system.

Capture and interpret ICMP (Internet Control Message Protocol) traffic between a guest VM and an external server.

2. Environment Configuration
Virtual Machine: Ubuntu Linux running on Oracle VirtualBox.

Network Setup: The VM is configured to use a network adapter that bridges to the host's Ethernet 7 interface.

The capture confirms that the network analyzer is properly observing the traffic flow. The consistent replies from 
the destination server prove that the virtual network is stable and the packet capture driver is accurately recording
data without loss.
